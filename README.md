# Lucid Suspension

The website for [Lucid Suspension](https://lucidsuspension.com) — a transparent, rider-first dirt bike fork and shock service.

## Tech stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript

## Getting started

Requires Node 18+.

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000)

## Project structure

- `src/app/(site)` — route segments and page components
- `src/app/components` — shared UI
- `src/app/lib` — config, design tokens, SEO, site config
